# semiscan

The repository aims to provide a reproducible code example for the paper accessible here: https://doi.org/10.1016/j.cose.2025.104681

### Setup
Create a virtual environment:
```
python3 -m venv .env
source .env/bin/activate
```
Install dependencies:
```
pip install --upgrade pip
pip install -r requirements.txt
```

### Usage
Run the framework:
```
python semiscan.py <path-to-data> <output-folder>
```
